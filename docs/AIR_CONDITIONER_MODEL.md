# AIR_CONDITIONER_MODEL.md

본 문서는 EMS에서 제어하는 에어컨의 역할과 제어 정책을 정의한다.

---

## 1. 에어컨의 역할

- 배터리 룸 온습도를 **조절하기 위한 제어 수단**
- 충·방전 판단의 직접적인 기준 장비는 아님

---

## 2. 제어 모드

### AUTO (자동)

- EMS가 온습도 기준을 참조하여 능동적으로 제어
- 냉방 / 난방 / 제습 / 풍량 등을 자동 조절

### MANUAL (수동)

- 엔지니어가 설정한 상태를 그대로 유지
- EMS는 자동 제어에 개입하지 않음

---

## 3. 충·방전과의 관계

- 에어컨 상태는 충·방전 판단에 직접 사용되지 않는다
- 충·방전 판단은 오직 온습도계 기준에 의해 수행된다

---

## 4. UI 표현 원칙

| 상태 | 표시 |
|---|---|
| 자동 제어 중 | 자동 제어 중 |
| 수동 모드 | 수동 제어 유지 중 |
| 통신 불가 / 고장 | 빨간색 경고 표시 |

본 문서는 에어컨 Service 및 UI 구현의 기준 문서로 사용된다.
